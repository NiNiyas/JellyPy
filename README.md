# JellyPy

This project is a fork of [Tautulli](https://github.com/Tautulli/Tautulli).

A python based web application for monitoring, analytics and notifications for [Jellyfin](https://jellyfin.org/).

## What it can do:

- Login to Jellyfin
- Displays
	- Libraries/Media
	- Activity
	- Users
    - Recently added
